## Built With
- Go: Back-end language
- Gin: Go web framework
- HTMX: Front-end framework
- TailwindCSS: Styling system


## Installations

### Go

1. ```go install .```
2. ```go run .```


### HTMX + TailwindCSS

1. ```pnpm install```
2. To run watch mode for TailwindCSS:
```pnpm tw```
3. To compile JS in watch mode:
```pnpm js```
